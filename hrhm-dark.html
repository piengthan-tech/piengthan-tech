คุณปิติกาญจน์ สิทธิเดช
กรรมการสิทธิมนุษยชนแห่งชาติ (กสม.)
"การส่งเสริมและคุ้มครองสิทธิมนุษยชนในพื้นที่จังหวัดชายแดนภาคใต้กับบทบาทของพระสงฆ์"
  • กสม. เป็นองค์กรอิสระที่ต้องอาศัยความร่วมมือจากทุกภาคส่วน — พระสงฆ์คือขวัญกำลังใจและที่พึ่งของชาวบ้าน
  • การจัดตั้ง "ศูนย์ประสานงานพุทธวิธีเพื่อสิทธิมนุษยชน" เมื่อ ๗ กันยายน ๒๕๖๘ เพื่อใช้แนวทางพุทธวิถีช่วยเหลือประชาชน
  • ความพิเศษรุ่นที่ ๒: นำ AI มาช่วยจัดการข้อมูล พระสงฆ์เชื่อมต่อกับศูนย์ประสานงานฯ ได้ทุกที่ทุกเวลา
  • กสม. สำนักงานภาคใต้ทำหน้าที่คนกลาง รวบรวมข้อเสนอแนะจากกิจกรรมส่งต่อรัฐบาล

คุณจุมพล ขุนอ่อน
รองเลขาธิการคณะกรรมการสิทธิมนุษยชนแห่งชาติ
"สิทธิมนุษยชนกับหลักพุทธศาสนา — เรื่องเดียวกัน"
  • หัวใจสิทธิมนุษยชน = เคารพศักดิ์ศรีความเป็นมนุษย์อย่างเท่าเทียม สอดคล้องกับหลักพรหมวิหาร ๔
  • พระสงฆ์มี ๓ บทบาทสำคัญ: ผู้รับฟัง · สะพานเชื่อมประสาน · ที่พึ่งทางใจและให้สติ
  • กสม. มาเพื่อเติมเต็ม "เครื่องมือ" — ความรู้สิทธิ กฎหมาย การติดต่อหน่วยงาน

ช่วงที่ ๓ · เช้า
พระราชวัชรญาณโมลี (เจ้าคุณชรัตน์)
รองเจ้าคณะภาค ๑๘ · เจ้าอาวาสวัดตานีนรสโมสร พระอารามหลวง
"บทบาทของพระสงฆ์ในการช่วยเหลือเยียวยาผู้ได้รับผลกระทบในพื้นที่ชายแดนใต้"
  • "ทำแก้วให้ว่าง" — เปิดรับวิธีคิดยุคใหม่ เพื่อนำหลักธรรมมาประยุกต์กับสิทธิมนุษยชน
  • หลักทิศ ๖ ของพระพุทธเจ้าคือรากฐานสิทธิมนุษยชน — ทุกคนทำหน้าที่สมบูรณ์ จะไม่เกิดการละเมิดสิทธิ
  • ธรรมะ + กฎหมาย + เทคโนโลยี = ความเข้มแข็งในการช่วยเหลือชุมชน
ช่วงที่ ๔ · เช้า
ผศ.ดร.สุชาติ เศรษฐมาลินี
กรรมการสิทธิมนุษยชนแห่งชาติ
"สิทธิมนุษยชนเรื่องใกล้ตัว — หลักการพื้นฐาน กฎหมาย กลไก"
  • สิทธิมนุษยชนไม่ใช่ "ไม้เมืองหนาว" — รับรองทั้งในรัฐธรรมนูญไทยและกติกาสากล หลักการหลัก ๔ ประการ: ศักดิ์ศรีความเป็นมนุษย์ · ความเสมอภาค · เสรีภาพ · ความยุติธรรม
  • กรณีพระสงฆ์ถูกกล่าวหา: ห้ามบังคับสึกก่อนศาลตัดสิน (สันนิษฐานว่าบริสุทธิ์ก่อน)
  • เมื่อมีเจ้าหน้าที่ขอตรวจสอบ: ขอดูบัตร → แจ้งเจ้าคณะ → อ้างกฎหมายที่ถูกต้อง
  • สันติภาพยั่งยืน = ความมั่นคง + กระบวนการยุติธรรมโปร่งใส เดินคู่กัน
กิจกรรมกลุ่ม · บ่าย
พระมหานภันต์ สนฺติภทฺโท ป.ธ.๙
ผู้ช่วยเจ้าอาวาสวัดสระเกศราชวรมหาวิหาร · ประธานมูลนิธิ สกพ.
"รู้ความเป็นมา รู้ความเป็นไป รู้จัก รู้ใจ รู้ร่วมแก้ไขด้วยธรรม"
  • โอวาทสมเด็จพระพุฒาจารย์: "เราตายได้ แต่พุทธศาสนาตายไม่ได้ ขอให้พระเป็นกลุ่มสุดท้ายที่ออกจากแผ่นดินนั้น"
  • กิจกรรมจำลองการไม่ถูกรับฟัง — ให้เวลาพระสงฆ์เล่าเรื่องเพียง ๑ นาทีแล้วตัดจบ เพื่อให้รู้สึกถึงความอึดอัดของผู้ถูกละเมิด
  • เป้าหมาย: ยังมีคนในสังคมที่เสียงความทุกข์ "ยังไม่ได้รับการเยียวยา" พระสงฆ์คือผู้ทำให้เสียงนั้นดังขึ้น

ดร.นพ.สกล สิงหะ
คณะแพทยศาสตร์ มหาวิทยาลัยสงขลานครินทร์
"ธรรมะ: ต้นทุนสำหรับเยียวยาตน คน และสังคม"
  • ผัสสะ vs อารมณ์: ความเจ็บปวดทางกายแก้ด้วยยา แต่อารมณ์ทุกข์เกิดจาก "เรื่องเล่า" ในใจ ต้องแก้ด้วยธรรมะและการปรับมุมมอง
  • กรณีผู้ป่วยมะเร็ง ๒ คน — คนหนึ่งมองว่า "มะเร็งเป็นสิ่งดีที่สุด" เพราะทำให้ครอบครัวกลับมารวมกัน พิสูจน์ว่าเรื่องเล่าต่างกัน ผลต่างกัน
  • ผู้เยียวยา = "ตัวประกอบยอดเยี่ยม" ไม่ใช่พระเอก แต่เอื้ออำนวยให้ผู้มีทุกข์เดินหน้าต่อไปได้
  • ใช้หลัก โพชฌงค์ ๗ และพรหมวิหาร ๔ เป็นฐานกระบวนการเยียวยา

  • AI เป็นเพียงเครื่องมือสนับสนุน บทบาทมนุษย์และจิตใจที่เมตตาไม่อาจถูกแทนที่
  • AI ช่วย: สร้างสื่อสรุปธรรมะ / จัดการข้อมูลผู้รับการเยียวยา / สื่อสารระหว่างเครือข่ายพระสงฆ์ชายแดนใต้
  • แพลตฟอร์มดิจิทัลให้พระสงฆ์ปรึกษา แลกเปลี่ยนประสบการณ์ และส่งต่อข้อมูลได้แม้อยู่คนละพื้นที่
  • ขอบเขตจริยธรรม: ไม่ขัดต่อพระวินัย รักษาความลับผู้รับบริการ อยู่ภายใต้การตัดสินใจของมนุษย์เสมอ

"ทักษะการฟังอย่างตั้งใจ (Active Listening) เพื่อการเยียวยา"
  • ๓ ระดับการฟัง: (๑) ฟังเพื่อตอบ → (๒) ฟังเพื่อเข้าใจ → (๓) ฟังเพื่อเชื่อมต่อ (ระดับที่พระสงฆ์ควรพัฒนา)

  • วิทยาลัยสงฆ์ปัตตานีเป็นฐานวิชาการ สนับสนุนองค์ความรู้และผลิตบุคลากร

ศูนย์อำนวยการบริหารจังหวัดชายแดนภาคใต้
"บทบาท หน้าที่ และอำนาจของ ศอ.บต. กับการอำนวยความยุติธรรมและเยียวยาผู้ได้รับผลกระทบ"
  • ศอ.บต. มีอำนาจบูรณาการทุกหน่วยงานในพื้นที่ ภารกิจหลัก ๓ ด้าน: อำนวยความยุติธรรม · เยียวยา · สร้างสันติสุข
  • ผู้มีสิทธิ์รับการเยียวยา: ผู้เสียชีวิต / บาดเจ็บ / พิการ / ได้รับผลกระทบด้านทรัพย์สิน
  • พระสงฆ์เป็น "ตัวกลาง" ช่วยผู้ได้รับผลกระทบในชุมชนเข้าถึงสิทธิเยียวยาจาก ศอ.บต.
